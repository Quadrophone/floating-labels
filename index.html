jQuery Floating Labels

Include the script and jQuery on your page, then add the "floating" class to any input or textarea element. Set the "placeholder" attribute to whatever you want to appear in the floating label.

<input type="text" class="floating" placeholder="Name">

Add this to your CSS:

.floating {
position:relative; z-index:99;
padding:10px;
}
.floating-label {
position:absolute;
z-index:999;
}

(Adjust the padding as needed)
